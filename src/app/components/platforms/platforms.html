Dù bạn làm việc trên hệ điều hành nào, Funput đều mang lại trải nghiệm gõ tiếng Việt đồng nhất, mượt mà và tin cậy nhất.

Sẵn sàng

macOS

Input Method native thực sự, tích hợp sâu vào hệ thống. Hiển thị thanh lịch trên Menu bar.

  • Hỗ trợ Apple Silicon & Intel
  • Không cần quyền admin (Tải bản .zip)
  • Đã ký Developer ID + Notarized
Tải bản .pkg / .zip

Windows

Chạy nền siêu nhẹ, xuất hiện tinh tế ở khay hệ thống (System Tray). Gõ mượt mà trên mọi ứng dụng.

  • Hỗ trợ Windows 10 & 11 (64-bit)
  • Phím tắt chuyển đổi nhanh: Ctrl + `
  • Giao diện Onboarding chào mừng trực quan

Linux

Addon Fcitx5 viết bằng C++ bọc lõi Rust. Hoạt động hoàn hảo trên cả môi trường X11 và Wayland.

  • Hỗ trợ kiến trúc amd64 và arm64
  • Tích hợp mượt mà với Fcitx5
  • Đóng gói dạng .deb cài đặt cực nhanh
